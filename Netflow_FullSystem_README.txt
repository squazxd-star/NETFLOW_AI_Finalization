 FOLDER STRUCTURE

------------------------------------------------------------
 INSTALLATION - Chrome Extension
------------------------------------------------------------

1. เปิด Google Chrome
2. พิมพ์ chrome://extensions ในช่อง URL แล้วกด Enter
3. เปิด "Developer mode" (มุมขวาบน)
4. กด "Load unpacked"
5. เลือกโฟลเดอร์ "extension" ที่อยู่ในแพ็คเกจนี้
6. Extension จะปรากฏในแถบเครื่องมือ Chrome
7. คลิกไอคอน Netflow AI เพื่อเปิด Side Panel

------------------------------------------------------------
 INSTALLATION - Native Host (Optional)
------------------------------------------------------------

Windows:
1. ติดตั้ง Node.js (https://nodejs.org) ถ้ายังไม่มี
2. เปิด Command Prompt ในโฟลเดอร์ native-host
3. รัน: node install.js
4. รีสตาร์ท Chrome

macOS:
1. ติดตั้ง Node.js ถ้ายังไม่มี
2. เปิด Terminal ในโฟลเดอร์ native-host
3. รัน: node install-mac.js
4. รีสตาร์ท Chrome

------------------------------------------------------------
 USAGE
------------------------------------------------------------

1. กรอกข้อมูลสินค้า (ชื่อ, รายละเอียด, รูปสินค้า)
2. เลือกเทมเพลต, สไตล์, ฉากพื้นหลัง
3. กด "สร้าง/อัปเดต Prompt" เพื่อให้ AI สร้างคำสั่ง
4. กด "เริ่มสร้างวิดีโอ" เพื่อเปิด Automation
5. ระบบจะสร้างภาพ + วิดีโอ + อัปโหลดอัตโนมัติ
